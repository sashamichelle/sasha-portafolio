Proyecto "Experiencia en Buses"

Desafío; optimizar y crear una solución a los problemas ya existentes del servicio de transporte en buses, encontrar las principales debilidades y proponer alternativas para mejorar el mismo.
¿Quién?


Estamos realizando este proyecto para las personas que usan frecuentemente el servicio de buses en Chile, considerando sus perspectivas y experiencias de uso.
¿Cómo?





Estamos estableciendo cuales son las principales problemáticas mediante diferentes métodos de investigación, como lo son Customer Journey Map, Story Telling, etc.
¿Porqué?

Para relizar este desafío debíamos plantearnos el motivo para plantear una solución, las personas solo tienen este medio de transporte para trasladarse desde una región a otra, muchas veces obligatorio.

Para quién esta dirigido

Realizamos con mi equipo de trabajo tres user personas para poder empatizar con los posibles usuarios que tiene el servicio, analizar como pueden reaccionar frente a una situación.

Si quieres saber más de nuestro usuario objetivo, accede a el StoryTelling realizado por mi.

Proyecto BP Digital

Realice desafío que consistia en mejorar o diseñar una página Web o Aplicación, a partir de allí nació la idea de mejorar la actual plataforma de la Biblioteca Pública Chilena.

Vista de la Home de la página web a la que esta dirigido el proyecto.Si quieres ver la página oficial puedes ver Aquí.
¿Quién?


Estamos realizando este proyecto para las personas que usan frecuentemente la página Web y aplicación Móvil, considerando que tiene que ser utilizado por todos los chilenos ya que es una plataforma gratuita.
¿Cómo?




Se estableció de acuerdo a las problemáticas y necesidades del público lector y no lector, que necesitan disponer de libros de forma fácil y rápida.Mediante encuestas, formularios y análisis de páginas actuales.
¿Porqué?




Para relizar este desafío debíamos plantearnos el motivo para plantear una solución,las personas necesitan disponer de archivos y incentivar a la lectura, no solo que sea por obligación de las curriculas escolares.

¿Cómo se ve la solución?

Realicé un prototipo de la aplicación móvil, donde se pueden ver los cambios realizados a la actual interfaz de la misma.
